# Sheher - Shared

Shared types and utilities used across the Sheher monorepo.

## Contents

- `types.ts` - Shared TypeScript interfaces and types
- `constants.ts` - Shared constants (future)
- `utils.ts` - Shared utility functions (future)

## Usage

Both frontend and backend can import from this package.

### Frontend (from `frontend/src`)
```typescript
import { Place, User } from '@sheher/shared/types';
```

### Backend (from `backend/src`)
```typescript
import { Place, User } from '@sheher/shared/types';
```
